Boomerang Learning (Bubble.io)

This was a project I've been working on by myself since February. It is an educational platform that gives AI-generated questions based on the Indiana State Curriculum, to students adaptively in order to target student weaknesses. Data on student progress is then reported back to teachers and parents to help them teach better.

This was all done with a No-Code Platform called Bubble.io, which utilizies very similar elements as HTML and CSS. My experience at Catapult greatly facilitated me in understanding the same styling properties, and gain an in-depth understanding of actual code.
